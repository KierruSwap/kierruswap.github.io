KierruSwap
Hello World! This is the version 1 for KierruSwap. Please bear with the UI, this is still not final. I attempted this project because I hate gas fees. In this tool, you can now buy a horse without paying high gas fees! Right now, this site is using a third party swapper that charge 1% fee, it goes directly to the third party. For the charge to use this service, it will be 1 MATIC flat rate that will be use in the future plans below.
Roadmap

  • Add MATIC topup feature.

  • NFT membership, no more fees! Plus get whitelisted on my upcoming NFT game :)
